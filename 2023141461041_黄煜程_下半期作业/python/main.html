Python 代码规范#

本节介绍 Python 代码规范, 以增强代码风格统一性与可读性. 本文档撰写具体参考 PEP 及 Google代码规范.

本代码规范分为三个级别, 分别为

  1. 强制: 要求任何项目必须实现的规范.

  2. 推荐: 鼓励项目使用的规范, 但项目应根据实际情况裁剪.

  3. 允许: 不鼓励也不拒绝的规范.

对于推荐和允许级别, 每个项目应该自行根据实际情况裁剪, 制定一个统一的规范.

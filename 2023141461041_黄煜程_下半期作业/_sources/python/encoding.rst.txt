.. _encoding:

编码
====

强制
----
- 所有 Python 源文件必须统一使用 UTF-8 编码格式保存.
- 禁止在文件中出现不可见或非法字符.

推荐
----
- 文件开头可显式声明编码(如 ``# encoding: utf-8``), 以增强可移植性.
- 编辑器和工具链应统一设置为 UTF-8 默认编码.

允许
----
- 特殊情况下可根据第三方库或历史原因保留其他编码，但需在注释中说明理由.

示例
----
.. code-block:: python

    # encoding: utf-8
    # 本文件采用 UTF-8 编码，支持中文注释
    print("你好，世界！")

空格
====

强制
----
- 除函数参数内无类型注释的赋值等号外, 所有二元运算符两侧必须加一个空格.
- 如果一个逗号所在同一行后还有内容, 则应当在该逗号后加一个空格.
- 字典键值对, 类型注解中的冒号后需要加一个空格, 但切片操作中的冒号后不加.
- 注释使用的 ``#`` 开头后跟一个空格. 类型注解所用箭头前后各一个空格.
- 每行代码末尾不应当有空格.
- lambda函数的参数与冒号之间不应该空格, 冒号后应该空一格

推荐
----
- 代码块内的空格应保持风格一致, 提升可读性.
- 对齐多行参数, 字典, 列表时可适当增加空格以增强视觉整齐.

允许
----
- 在复杂表达式中, 为了突出运算优先级, 可适当减少或增加空格, 但需保证一致性和可读性.

示例
----
.. code-block:: python

    result = a + b * c
    def func(a, b=0):
        pass

    # 这个函数用来打招呼
    def greet(name: str = 'Judy') -> str:
        return 'Hello, {}'.format(name)

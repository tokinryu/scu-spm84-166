.. _punctuation:

标点符号
========

强制
----
- 除特殊语法外, 禁止在 Python 代码中出现顿号和分号, 一律使用逗号.
- 禁止在一行内用分号分隔多条语句.
- 列表, 字典, 元组, 函数参数等展开格式, 最后一个元素末尾不加逗号.
- 代码中应规范使用英文标点, 避免中英文混用.

推荐
----
- 如无特殊需要(如doctest), 否则鼓励只使用逗号和句号.

允许
----
- 特殊业务或兼容性需求下可适当调整标点用法, 但需注释说明.

示例
----
.. code-block:: python

    # 正确
    items = [1, 2, 3]
    def foo(a, b, c):
        pass

    # 错误
    x = 1; y = 2
    data = [1, 2, 3,]
